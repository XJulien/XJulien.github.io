发表于 2019-03-08 | 分类于 Java | 阅读次数：

目录结构

代码层结构，根目录com.springboot
  • domain
    • 实体类
    • 适用于jpa项目（即基于hibernate操作数据库）
  • pojo
    • 实体类
    • 适用于mybatis项目
  • repository（dao）
    • 数据接口访问层
    • 适用于jpa项目
  • mapper（dao）
    • 数据接口访问层
    • 适用于mybatis项目
  • service
    • 数据服务接口层
    • com.springboot.service.impl（Eclipse格式）
    • com.springboot.serviceImpl（Idea格式）
  • controller
    • 前端控制器
  • utils
    • 工具类
  • constant
    • 常量接口类
  • config
    • 配置信息类
  • dto
    • 数据传输对象
    • 用于封装多个实体类（domain）之间的关系，不破坏原有的实体类结构
  • vo
    • 视图包装对象
    • 用于封装客户端请求的数据，防止部分数据泄露，保证数据安全，不破坏原有的实体类结构
资源文件结构，根目录src/main/resources
  • mapper
    • mybatis映射文件
  • mapper/config
    • mybatis配置文件
  • config
    • 配置文件(.properties/.json等)
  • i18n
    • 国际化语言包
  • templates
    • 视图模板目录
  • static
    • 前端静态资源存放(js/css/image)
  • META-INF/spring
    • spring.xml

核心注解

  • @SpringBootApplication
    • 用于Spring Boot主类上，标识这是一个Spring Boot应用，用来开启Spring Boot的各项能力
    • 即 @SpringBootConfiguration、@EnableAutoConfiguration、@ComponentScan这三个注解的整合，可以使用这三个代替
  • @EnableAutoConfiguration
