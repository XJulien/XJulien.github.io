Java泛型

发表于 2019-01-27 | 分类于 Java | 阅读次数：

#####泛型

即参数化类型，将类型由原来的具体类型参数化，类似于方法中的变量参数，此时类型也就是类型形参

泛型类
  • 基础结构

  • 泛型标识符可以为任意字符，一般写为T
  • 实例化泛型类时，必须指定T的具体类型
  • 类型参数必须为类类型，不能为简单类型
  • 在实例化时如果不传入泛型类型实参，则在泛型类中使用泛型的方法或成员变量定义的类型可以为任意类型
  • 不能对确切的泛型类型做instanceof操作
泛型接口
  • 基础结构

  • 实现泛型接口

    • 若未传入泛型类型实参，则在实现接口时，需将此实现类同样通过泛型类的方式声明

    • 若传入了泛型类型实参，则在所有使用泛型的地方都要替换成传入的实参类型

泛型通配符
  • 上边界限定通配符
    • 通过使用 <? extends Parent> 形式的通配符，实现泛型的向上转型
    • 泛型可以使用一切基类及其子类
    • 不能往里存，只能往外取
  • 下边界限定通配符
    • 通过使用<? super child>，实现超类型的通配符
    • 泛型可以使用一切基类及其父类
    • 不影响往里存，但往外取只能放在Object对象里
  • 无边界通配符
    • 使用单独的问号<?>，表示不知道是哪种类型
    • 不可以向其中添加对象
泛型方法

  • 基础格式

  • 使用泛型方法所在的类并不一定是泛型类

  • 泛型方法必须得在返回值之前添加泛型参数标识，可以返回多个

  • 调用泛型方法时可以不显示地传入泛型参数，编译器会使用参数类型推断
