Dart基础学习

发表于 2018-05-16 | 分类于 Flutter
基础规则
  • 赋值
    • 只能把对象赋值给变量，每个对象都是类的实例
    • 数字、方法、null值都是对象
    • 所有对象继承自Object类
  • 声明
    • 变量时需要指明变量类型
    • 未指明的话则会变成特殊的dynamic
    • 任何变量的初始值为null，null类型的初始值也是null
  • 结构
    • 支持顶层方法(main()方法)
    • 支持类级别或对象级别的方法
    • 还可以在方法内部新建方法（嵌套方法或本地方法）
    • 支持顶层变量
    • 支持类级别或对象级别的变量，实例变量被称为域或者属性
    • 无public、 protected、 private，但是使用_下划线表示private
    • 标识符（变量名或方法名等）可以以字母或下划线_开头
    • 单引号与双引号无区别
  • 代码检查
    • Dart会在运行前对所有代码进行语法分析
    • Dart工具能报告2类问题：警告warning和错误error

user表示当前用户名

localhost为主机名称

～表示当前在～目录中，普通用户为/home/user, 超级用户为/root

NPM使用

发表于 2018-04-18 | 分类于 Web
安装工具

在安装nodeJS之后即可使用npm，全称：Node Package Manager

  • brew install node
（可选）CNPM

如果使用npm安装包文件时网速较慢，可以安装阿里的镜像cnpm，之后可以使用cnpm代替npm

  • npm install -g cnpm --registry=https://registry.npm.taobao.org
项目初始化

npm init [-y]

  • 在当前项目中生成package.json文件，加上-y则可以直接全部使用默认设置

ES6学习

发表于 2018-04-18 | 分类于 Web

ES6

声明
  • var 全局变量
  • let 块级本地变量
    • let a = 1;
    • 声明作用域、简化内部函数代码
    • 且不会在全局创造属性
  • const 常量
    • 一旦赋值，不可改变
  • 结构赋值
    • let [a,b,c]=[1,2,3];
    • let [a,[b,c],d]=[1,[2,3],4];
    • let [a,b="y"]=['x']; 解构的默认值
