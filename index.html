Java设计模式原则

发表于 2018-11-14 | 分类于 Java

单一职责原则

类的职责要单一，不能将太多的职责放在一个类中（高内聚，低耦合）

  • 降低复杂度以减少职责，从而提高可复用性
  • 提高可读性与维护性
  • 降低维护引起的风险性

开闭原则

对扩展开放，对修改关闭（设计模式的核心原则）

  • 尽量通过扩展已有实体，添加行为，抽象层模块不得修改
  • 通过抽象化实现开闭原则
  • 找到可变因素，并将其封装起来

Java基础

发表于 2018-10-14 | 分类于 Java

Java基础知识点总结，仅供复习时使用

数据类型
  • 基本数据类型
    • byte
      • 8位、有符号 整数
      • 最小值是-128(-2^7)
      • 最大值是127(2^7-1)
      • 默认值是0
      • 占用空间是int的四分之一
      • 包装类 Byte
    • short
      • 16位、有符号 整数
      • 最小值是-32768(-2^15)
      • 最大值是32767(2^15 - 1)
      • 默认值是0
      • 占用空间是int的二分之一
      • 包装类 Short

安装工具

在安装nodeJS之后即可使用npm，全称：Node Package Manager

  • brew install node
（可选）CNPM

如果使用npm安装包文件时网速较慢，可以安装阿里的镜像cnpm，之后可以使用cnpm代替npm

  • npm install -g cnpm --registry=https://registry.npm.taobao.org
项目初始化

  • 在当前项目中生成package.json文件，加上-y则可以直接全部使用默认设置
