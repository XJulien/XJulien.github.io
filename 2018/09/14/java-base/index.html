Java基础

发表于 2018-09-14 | 分类于 Java | 阅读次数：

Java基础知识点总结，仅供复习时使用

数据类型
  • 基本数据类型
    • byte
      • 8位、有符号 整数
      • 最小值是-128(-2^7)
      • 最大值是127(2^7-1)
      • 默认值是0
      • 占用空间是int的四分之一
      • 包装类 Byte
    • short
      • 16位、有符号 整数
      • 最小值是-32768(-2^15)
      • 最大值是32767(2^15 - 1)
      • 默认值是0
      • 占用空间是int的二分之一
      • 包装类 Short
    • int
      • 整数默认类型
      • 32位、有符号 整数
      • 最小值是-2,147,483,648(-2^31)
      • 最大值是2,147,485,647(2^31 - 1)
      • 默认值是0
      • 包装类 Integer
    • long
      • 64位、有符号整数
      • 最小值是-9,223,372,036,854,775,808(-2^63)
      • 最大值是9,223,372,036,854,775,807(2^63 -1)
      • 默认值是0L
      • 包装类 Long
    • float
      • 单精度、32位的浮点数
      • 默认值是0.0f
      • 不能用于表示精确的值
      • 包装类 Float
    • double
      • 浮点数默认类型
      • 双精度、64位的浮点数
      • 默认值是0.0d
      • 不能用于表示精确的值
      • 包装类 Double
    • boolean
      • true 或者 false
      • 默认false
      • 包装类 Boolean
    • char
      • 单一的16位Unicode字符
      • 最小值是'\u0000'（即为0）
      • 最大值是'\uffff'（即为65,535）
      • 默认值是'\u0000'（即为0）
      • char数据类型可以储存任何字符
      • 使用单引号声明
      • 包装类 Character
  • 引用数据类型
    • Array
      • 声明
        • dataType[] arrayRefVar;
        • dataType[] arrayRefVar = new dataType[arraySize];
        • dataType[] arrayRefVar = {value0, value1, ..., valuek};
      • 内置方法
        • .length
    • Class
    • Interface
  • 常量
    • 使用final关键字声明
    • 一般使用大写作为标识符
  • 类型转换
    • 规则
      • boolean不能转换
      • 转换类型应该是相关类的对象
      • 将大容量类型转小容量类型，需强制转换
      • 转换过程可能会导致溢出或损失精度
      • 浮点转整数，会舍弃小数，而不是四舍五入
    • 自动类型转换
      • 转换钱的数据类型位数要低于转换后的数据类型
    • 强制类型转换
      • 转换的数据类型必须兼容
      • 格式为：(type) value
    • 隐含强制类型转换
      • 整数默认为int
      • 浮点不存在此情况，在定义时必须在数字后跟上F或f
变量
  • 声明
    • 变量在使用前必须声明
    • 声明格式为type identifier [ = value][, identifier [= value] ...] ;
  • 局部变量
    • 声明在方法、构造方法、或语句块中
    • 局部变量在执行完成后，将会被销毁
    • 访问修饰符不能用于局部变量
    • 只在声明局部可见
    • 局部变量时在栈上分配的
    • 局部变量没有默认值，必须初始化！
  • 实例变量
    • 声明在类中，但在方法、构造方法、语句块之外
    • 随着对象被实例化而确定
    • 在对象被创建时创建、对象被销毁时销毁
    • 应保证被使用
    • 可以声明在使用前或使用后
    • 可以被访问修饰符修饰
    • 具有默认值，可以在声明时指定，或在构造方法中指定
    • 在静态方法或其他类中，应使用完全限定名访问
  • 静态变量
    • 类中以static声明， 但必须在方法、构造方法、语句块之外
    • 无论创建了多少对象，类值拥有静态变量的拷贝
    • 一般被声明为常量
    • 在第一次被访问时创建，程序结束时销毁
    • 可以被访问修饰符修饰，一般被声明为public
    • 具有默认值，和实力变量相似
    • 通过完全限定名访问
修饰符
  • 访问修饰符
    • default(即缺省，未添加)
      • 在同一包内可见
      • 类、接口、变量、方法
    • private
      • 在同一类内可见
      • 变量、方法
    • public
      • 所有类可见
      • 类、接口、变量、方法
    • protected
      • 对同一包内的类和所有子类可见
      • 变量、方法
    • 访问控制与继承
      • 父类为public，子类也必须为public
      • 父类中声明为protected， 子类为protected或public
      • 父类不可为private
  • 非访问修饰符
    • static
      • 静态变量
        • 用于声明独立与对象的静态变量，无论实例化多少对象，静态变量只有一份拷贝
      • 静态方法
        • 用于声明独立与对象的静态方法，不能调用类的非静态变量
    • final
      • 变量
        • 一经赋值，对象引用无法改变
      • 方法
        • 可以被子类继承，但是不能被子类修改
      • 类型
        • 无法被继承
    • abstract
      • 抽象类
        • 无法实例化，仅用于对类进行扩充
        • 与final冲突
        • 一旦存在抽象方法，则必须声明为抽象类，可以不包含抽象方法
        • 可以包含非抽象方法
      • 抽象方法
        • 没有方法的实现
        • 不能被声明为final或static
        • 子类必须实现所有抽象方法，除非子类也是抽象类
        • 以分号结尾
    • synchronized
      • 用于实例变量
      • 序列化时会被跳过
    • volatile
      • 多线程访问时，会强制重新读取
      • 可能为null

循环
  • while
    • while( expression ) { sentences }
    • 条件成立则执行
  • do...while
    • do{ sentences }while(expression)
    • 首先执行一次，然后根据条件是否成立来执行
  • for
    • for(int i=0;i<10;i++){ sentences }
    • for(String name : names){ sentences }
  • break
    • 跳出当前循环语句块
  • continue
    • 跳转到下一次循环
分支

    • if(expression){ sentences }
    • else{ sentences }

  • switch
    • swtich(expression){}
    • case 满足条件

方法
  • 定义
    • 结构
      • 修饰符
        • 可选
        • 定义方法返回访问类型
      • 返回值类型
        • 定义方法返回值类型
          • 无返回值则使用void
      • 方法名
        • 小驼峰式命名规则
        • 与参数共同构成方法签名
      • 参数
        • 需要有参数类型
        • 可选
      • 方法体
        • 定义方法实现的功能
      • 返回值
    • 重载
      • 方法同名，参数不同
    • 构造方法
      • 与类名相同
    • 可变参数
      • typeName... parameterName
      • 一个方法仅可指定一个可变参数
      • 必须是最后一个参数
    • finalize()
      • GC前调用方法
      • 对象不一定会被回收
