Webpack入门使用

发表于 2018-05-22 | 分类于 Web | 阅读次数：

前置技能

  • npm
  • ES6模块化相关知识

构建环境

  • 安装

npm i -g webpack

安装工具，在终端中输入webpack -v成功返回版本号则表示已安装成功

  • 配置文件

touch webpack.config.js

webpack在当前项目的配置文件

  • 目录结构
    • /
      • dist 项目发布目录，在执行打包压缩等操作后，会将生成的文件发布到此目录
      • src 过程中的开发引入目录
      • public 开发过程中，index.html根文件，以及图片等资源会存放到此目录
      • node_modules 执行npm i 会将package.json中的依赖安装到此目录中
      • webpack.config.js webpack在此项目中的配置文件
      • package.json 项目依赖文件
      • package-lock.json 项目依赖安装情况，此为npm i安装依赖后自动生成，无需理会

使用

  • 使用babel，ES6转ES5

    • 依赖
      • babel-core 核心库，真实的转换规则都在这里
      • babel-loader 用于通过webpack将ES6转ES5
      • babel-preset-env 用于检测自动检测当前ES版本，并将调用核心库将其转为ES5
