Dart基础学习

发表于 2018-05-16 | 分类于 Flutter | 阅读次数：
基础规则
  • 赋值
    • 只能把对象赋值给变量，每个对象都是类的实例
    • 数字、方法、null值都是对象
    • 所有对象继承自Object类
  • 声明
    • 变量需要指明变量类型
    • 未指明的话则会变成特殊的dynamic
    • 任何变量的初始值为null，null类型的初始值也是null

  • 结构
    • 支持顶层方法(main()方法)
    • 支持类级别或对象级别的方法
    • 还可以在方法内部新建方法（嵌套方法或本地方法）
    • 支持顶层变量
    • 支持类级别或对象级别的变量，实例变量被称为域或者属性
    • 无public、 protected、 private，但是使用_下划线表示private
    • 标识符（变量名或方法名等）可以以字母或下划线_开头
    • 单引号与双引号无区别
  • 代码检查
    • Dart会在运行前对所有代码进行语法分析
    • Dart工具能报告2类问题：警告warning和错误error
内置类型

  • 类型检查

    • if(1 is num){}
    • if('s' is String){}

  • numbers

    • num 包括：
      • int整数
      • double双精度浮点数
    • 内置函数
      • .abs()
      • .ceil()
      • .floor()
    • 类型转换
      • var one=int.parse('1');
      • .toString()
    • 数字字面量为编译时常量

  • strings

    • 是UTF-16编码单元的序列，可使用单引号或双引号括起来

    • 字符串拼接

      • var s1 = '这是 ' '字符串' " 拼接";

      • var s2 = '这是字符串 '+ '拼接';

    • 字符串追加r会保持原样输出，不会进行转义r'hi \n, yeah!'

    • ${expression}表达式，用于直接输出结果

    • 内置函数

      • .toUpperCase()
      • .toLowerCase()

  • booleans

    • 仅当值为true时布尔值为真
    • 否则，其他都为假

  • lists

    • 即arrays
    • 使用角标获取元素
    • 使用const生产编译时常量的值，从而使得lists无法修改
    • 如果需要类型检查，则使用<String>['Seth', 'Kathy', 'Lars']
    • 内置函数
      • .add()
      • .addAll([])
      • .length
      • .indexOf()
      • .removeAt()
      • .clear()
      • .sort((a,b) => a.compareTo(b));

  • maps

    • 键值对的形式

    • 赋值

      • var myMap1={"one": "1", "two": "2"};

      • var myMap2={1: "one", 2: "tow"};

      • 如果需要类型检查，则使用<String, String>{'index': 'HomePage', 'others': 'OhterPages'}

    • 内置函数

      • .containsKey()
      • .remove()
      • .keys
      • .values
      • .length
      • .putIfAbsent()

  • runes

    • 字符串的UTF-32编码格式
    • 用于表情等特殊字符

  • symbols

集合
  • 额外数据类型Sets
    • 是一种无序集合，因此也不发保护重复的数据
    • 同样无法通过索引来获取数据
    • 可以通过new Set.from(['X', 'Y'])来赋予初始值
    • 内置函数
      • .add()
      • .addAll([])
      • .length
      • .remove()
      • contains()
      • containsAll()
      • .intersection()获取交集
  • 同样包括Lists和Maps
  • 通用方法
    • .isEmpty
    • .forEach()
      • 在Map上使用时，会带有2个参数(k,v)
    • .map()
      • 该对象是懒求值的，只有访问是才会调用
      • 可以使用.toList()或则.toSet()来强制立刻执行
    • .where()
      • 返回所有满足特定条件的数据
    • .any()
      • 判断是否有满足特定条件的数据
    • .every()
      • 判断是否所有都满足特定条件
声明

  • 赋值

    • var name = 'Bob';
    • 未声明变量类型，默认为dynamic

  • 默认值

    • 未初始化，则默认值为null

  • 可选类型

    • 指定变量类型
    • String name = 'Bob';

  • 常量

    • const声明，是一个编译时的常量

    • 值必须得是编译时常量，即不可为函数返回值或实例化对象

  • 固定值

    • final声明，只可赋值一次
    • final name = 'Bob';
    • final String name = 'Bob';
内置函数
  • assert()
    • 断言函数，当断言失败时函数立刻终止
    • assert(1 == 1);
    • 生产模式中，将会忽视断言函数
    • 追加第二个参数用于自定义报错信息
  • print()
    • 打印
函数

  • 入口函数main()

  • 返回

    • 无返回值时，函数返回null
    • 可选返回类型声明
    • void FuncName()

  • 传参

    • 可选传参类型声明

  • 箭头函数

    • =>expr;等价于{return expr;}
    • 同样支持匿名函数

  • 函数别名(待补充)

  • 闭包

    • 函数返回函数类型
    • 函数同样也是对象

  • 可选参数

    • 命名可选参数
      • 函数声明中使用对象的形式命名{}
    • 位置可选参数
      • 函数声明中使用数组的形式命名[]
操作符与流程控制

  • 取整

    • ~/
    • print(3~/2);

  • 级联

    • ..

  • 判断

    • 条件为true则通过判断
    • 在checked模式下，如果是非bool值会抛出异常
    • 在production模式下，非 bool值会被反应成false

  • 循环

    • for

    • forEach

      • 针对容器遍历

    • for-in

      • 正对容器遍历

    • while

    • do-while

  • Switch And Case

    • 参数为num或String

    • 如果分句为空，可以参略break

    • 如果不为空，则必须加break，否则将会抛出异常

    • 可以使用continue不抛出异常

异常

  • 可以抛出非空对象

    • throw new ExpectException('值必须大于0！');
    • throw '值必须大于0！';

  • 可以抛出多个类型的Exception，若catch分句未指定类型，则处理所有异常

  • 可以使用finally使得某些代码始终运行

类与对象

  • 定义

    • 使用class关键字定义类
    • 使用new关键字创建对象
    • 未显示定义构造函数，则默认一个空的构造函数

  • 构造函数

    • 实例化对象时可以省略new关键字

    • 声明构造函数

      • 与对象同名的方法

      • 使用this关键字修改对象属性

    • 命名构造函数

      • 格式为Class.name(var param)

      • 使用冒号初始化变量

      • 可以使用简写: Point.fromeList(var list):this(list[0], list[1], list[2]);

    • 常量构造函数

      • 使用const定义编译时常量对象
      • const ImmutablePoint(this.x, this.y);

    • 工厂构造函数

  • Getters And Setters

    • get
    • set

  • 接口

    • 使用implements来调用类，此时不会继承这个类的实现
    • 类只可继承一个接口

  • 抽象类

    • 使用abstract来实现抽象类
    • 抽象类无法实例化
    • 抽象方法不需要关键字，且直接以分号结束
    • 使用关键字implements来实现抽象类

  • 继承

    • 使用extends继承类
    • 在子类中调用父类的方法，使用super.来实现

  • 静态方法

  • 操作

    • 使用?.避免null
    • 构建同样的编译时常量对象时，两者instance相同
    • 可以使用.runtimeType来获取运行时的变量类型
泛型

  • 使用类型来进行限制

    • var names = new List<String>();

  • 使用一个字母代替类型参数

    • E, T, S, K, V等
