Vue基础

发表于 2018-06-08 | 分类于 Web | 阅读次数：
指令
  • v-text

渲染dom内容

  • v-bind
    • 渲染dom属性, 使用此语法可用于添加dom的动态属性
    • 缩写使用:代替
    • class
      • 对象，控制开关
      • 数组，可使用三元表达式
      • 二者可以混用
    • style
      • 对象，键值对，多词使用驼峰式
      • 数组，追加对象值
      • 多值，只会渲染最后一个浏览器支持的值
  • v-if
    • 条件语句
    • 支持v-else以及v-else-if
  • v-show

切换是否显示

  • v-for
    • 循环语句
    • 数组
      • 建议使用of语句
      • 获取索引， (item, index) of items
    • 对象
      • 可获取对象属性的值
      • 获取键，(value, key) in object
      • 获取索引， (value, key, index) in object
    • 应当追加一个key值
  • v-on
    • 监听事件
    • 缩写使用@代替
  • v-model

表单的双向绑定

  • v-once

一次性插值

  • v-html

不作转译，危险情况，可能导致XSS攻击

生命周期
  • beforeCreate
  • created
  • beforeMount
  • mounted
  • beforeUpdate
  • Updated
  • beforeDestory
  • destoryed

修饰符
  • 表单修饰符

    • .lazy

      延迟更新，焦点离开文本框再做追加

    • .number

      内容自动转为数字

    • .trim

      自动过滤前后空字符

  • 事件修饰符

    • .stop

      阻止冒泡并触发回调，但是不会拦截默认行为

    • .prevent

      阻止默认行为并触发回调，但是不会阻止冒泡

    • .capture

      追加事件捕获

    • .self

      仅本身元素触发事件时触发回调

    • .one

      仅触发一次

    • .passive

      直接触发事件，与.prevent相冲突

  • 按键修饰符

    • 原生数字码

      指定键盘数字, 如@keyup.13

    • 键盘事件别称

      • .enter
      • .tab
      • .delete
      • .esc
      • .space
      • .up
      • .down
      • .left
      • .right
  • 系统修饰符
    • .ctrl
    • .alt
    • .shift
    • .meta
    • .exact
    • .left
    • .right
    • .middle
方法
  • computed
    • 有缓存，依赖发生改变才会重新触发，否则会直接返回之前的值
    • 默认为getter, 可自定义setter
  • method
    • 无缓存，始终重新计算
  • watch
    • 侦听属性，触发方法
Prop
  • 命名

  • 绑定

动态绑定数据

  • 单向

    • String

    • Boolean
    • Function
    • Object
    • Array
    • Symbol
  • style与class

会自动进行合并操作

  • .sync

扩展为自动更新父组件属性
