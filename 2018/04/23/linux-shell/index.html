Linux常用命令

发表于 2018-04-23 | 分类于 Linux | 阅读次数：
命令前缀格式

user@localhost:~$

user表示当前用户名

localhost为主机名称

～表示当前在～目录中，普通用户为/home/user, 超级用户为/root

$为用户标识符，表示普通用户，#表示超级用户

目录结构
  • / 最顶级根目录
    • bin 命令保存目录
    • usr 包目录
    • etc 配置文件目录
    • home
    • lib
    • mnt
pwd

显示用户当前所在路径

ll

需要保证存在别名alias ll = "ls -al"

别名就是将一些常用的命令直接使用缩写的方式声明即可

ls即list的缩写

参数-a表示显示隐藏文件

参数-l表示以列表的形式显示

bash中自定义别名可以在 ~/.bash_profile文件中编辑

zsh中则是 ~/.zshrc

cd

用于切换目录 即change directory的缩写

后面可以追加相对路径或绝对路径

如cd /usr/local或cd ../usr/local

可以使用缩写格式

  • cd ~切换到当前用户的/home/user目录下
  • cd -返回之前的目录
  • cd ..返回上级目录
rm -fr

用于强制删除目录

-f表示直接删除 不加的话会问你是否确定删除

-r表示目录 如果仅删除文件则不需要加

mkdir

创建目录 即make directory的缩写

cp

复制

mv

前切

cat

查看文件内容

chown

修改文件用户/用户组

chmod

修改文件权限

sudo

以root的身份执行命令 没事千万别用 比如需要删除某个目录 但是那个目录现在正在被占用 就会提示删除失败 此时使用 sudo rm -fr XXX 则可以强制删除

top
ps
netstate
vi/vim
less
cat
tail
grep
brew

mac常用包管理软件 类似的还有Ubuntu的apt Centos的yum 这些其实都只是用于安装软件的工具

brew update

更新brew仓库

brew upgrade

升级软件

brew remove

删除软件

brew list

列出软件

brew cleanup

清理不需要的下载包
