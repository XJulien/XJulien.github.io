Docker命令

发表于 2018-04-17 | 分类于 Linux | 阅读次数：

镜像加速

sudo vim /etc/docker/daemon.json

docker run

在容器内运行程序 sudo docker run -ti 交互式执行 sudo docker run -d 后台模式 sudo docker run -p 指定端口 sudo docker run --name xxx 指定容器名称 sudo docker run -v /opt/nginx/www:/www:[rw|ro] 挂载容器目录

docker ps

显示运行中的容器 sudo docker ps -a 显示所有容器 sudo docker ps -l 显示最后创建的容器 sudo docker ps -aq 显示所有容器id

docker port

查看端口映射

docker logs

查看容器内部标准输出 sudo docker logs -f 实时追加显示日志

docker top

查看容器内部运行进程

docker inspect

查看容器配置与状态信息

docker stop

停止容器

docker start

启动停止容器

docker restart

重启容器

docker rm

删除容器 sudo docker rm $(sudo docker ps -aq) 删除所有容器

docker images

列出本地镜像

docker search

搜索镜像

docker pull

下载镜像到本地
