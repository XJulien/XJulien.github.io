NPM使用

发表于 2018-04-18 | 分类于 Web | 阅读次数：
安装工具

在安装nodeJS之后即可使用npm，全称：Node Package Manager

  • brew install node
（可选）CNPM

如果使用npm安装包文件时网速较慢，可以安装阿里的镜像cnpm，之后可以使用cnpm代替npm

  • npm install -g cnpm --registry=https://registry.npm.taobao.org
项目初始化

npm init [-y]

  • 在当前项目中生成package.json文件，加上-y则可以直接全部使用默认设置

安装

已有项目，则到项目根目录下执行npm install，则会将package.json中的所有依赖都安装

  • 全局安装
    • npm i -g <pkg>
    • 安装到/usr/local/lib/node_modules目录中
    • 此目录在安装node时会自动追加到环境变量中，因此所有此目录下的文件可以直接执行
  • 本地安装
    • dependencies
      • npm i -S <pkg>
      • 在
    • devDependencies
      • npm i -D <pkg>

注：所有的<pkg>安装后面都可以跟一个版本号，格式为<pkg>@<version>，如webpack@4.0.0

查看
  • npm ls <pkg>

查看当前项目包的信息

  • npm info <pkg>

查看当前项目包的详细信息

  • npm ls

查看当前项目中所有的包信息

  • npm view <pkg> versions

查看某个包的所有远程版本号

更新
  • npm update

将当前项目中的所有包更新

  • npm update <pkg>@<version>

将当前项目中的包更新到指定版本

  • npm update -g

更新全局包

卸载
  • npm uninstall <pkg>

卸载当前项目中的包

清理缓存
  • npm cache clean
