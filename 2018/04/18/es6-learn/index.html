ES6

声明
  • var 全局变量
  • let 块级本地变量
    • let a = 1;
    • 声明作用域、简化内部函数代码
    • 且不会在全局创造属性
  • const 常量
    • 一旦赋值，不可改变
  • 结构赋值
    • let [a,b,c]=[1,2,3];
    • let [a,[b,c],d]=[1,[2,3],4];
    • let [a,b="y"]=['x']; 解构的默认值

字符串模版

箭头函数
  • 比函数表达式更短
  • 不绑定this、arguments、super或new.target
  • 不能用作构造函数
class
  • class 声明
  • constructor 构造函数
  • extends 继承
  • super 调用父级方法
  • static 静态方法（只能在类上进行）
Generator
  • function命令与函数名之间有一个星号。
  • 函数体内部使用yield语句定义不同的内部状态。
Set

它类似于数组，但是成员的值都是唯一的，没有重复的值。Set本身是一个构造函数，用来生成Set数据结构。

Promise

用于解决js的回调地狱问题

无疑，回调方式是最非人类的，一旦嵌套过多，则很难去理解其中的逻辑

参考来源：async/await 语法到底好在哪里？

    • import/export
    • 用法较多
